## Options
- [Overleaf](https://www.overleaf.com/project) (industry grade, free plan is limited)
	- Single email for collaboration (might get banned)
	- No GitHub integration
	- [Open in Overleaf Badge](https://gist.github.com/sugatoray/5c9ec0d837bb0cc98bc7d98544a91c6f) (one-way only)
- [TexPage](https://www.texpage.com/console/) (very similar to Overleaf, no GitHub integration)
	- Many can collaborate
- [Paperia](https://www.papeeria.com/me) (free, bad UI)
	- GitHub integration
- [LaTeX workshop](https://github.com/James-Yu/LaTeX-Workshop) (hardcore, less ease of use)
	- Requires a [LaTeX distribution](https://github.com/James-Yu/LaTeX-Workshop/wiki/Install)
- [TeXLyre](https://texlyre.github.io/texlyre/#yjs:g0egt5bu2pusmw3jhtielp&file:%2Farxiv-style-master%2Ftemplate.tex) (beautiful UI and styling)
	- Most balanced solution
	- GitHub integration
